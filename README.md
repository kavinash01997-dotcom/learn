# Linux Practice Repository

## Overview

This repository is created for learning and testing purposes. It contains practical implementations and configuration examples related to Linux system administration, networking, and server management.

## About This Repository

The main objective of this repository is to practice and document essential Linux commands and real-world server configuration tasks. It includes firewall management, file sharing services, FTP setup, containerization, and networking configurations.

## Topics Covered

* Firewall configuration using **iptables**
* Firewall management with **firewalld**
* Samba server setup and configuration
* VSFTPD (FTP Server) configuration
* Docker installation and basic container management

## Sample Command

```bash
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
```

The above command allows incoming TCP traffic on port 80 (HTTP).

## Purpose

This repository serves as:

* A personal Linux lab environment
* A reference for system administration tasks
* A practice space for server configuration and troubleshooting
